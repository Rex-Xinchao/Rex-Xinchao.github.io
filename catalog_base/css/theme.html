                # CSS 实现主题切换

                • 在 html 的 body 标签上添加自定义属性
                <body data-theme="light">
  <div id="app"></div>
</body>
                • 定义多套配色和样式的主题 css
                body[data-theme='light'] {
  @fontColor: #2c3e50;
  @borderColor: rgba(0, 0, 0, 0.06);
  @bgColor: #fff;
  @thumbColor: #ccc;
}
body[data-theme='dark'] {
  @fontColor: #2c3e50;
  @borderColor: rgba(0, 0, 0, 0.06);
  @bgColor: #fff;
  @thumbColor: #ccc;
}
                • 通过 js 来切换自定义属性实现主题的更换
